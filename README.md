# AI-Scripts
A collection of scripts relating to artificial intelligence. They are written in python and independent of each other. Used for studying terms in artificial intelligence from a low level perspective.

# AI Algorithms Repository

This repository contains five independent AI scripts:

1. **MNIST Classifier with PyTorch**
2. **Backpropagation Algorithm**
3. **Genetic Algorithm**
4. **Tabu Search**
5. **Clustering with Agglomerative Clustering**

Each of these scripts has additional pdf documentation next to it. For more detailed information, visit those pdfs.

## 1. MNIST Classifier with PyTorch

### Purpose
Classifies handwritten digits from the MNIST dataset using a neural network and compares three optimization algorithms: **SGD**, **SGD with momentum**, and **ADAM**.

### Usage
```bash
pip install torch matplotlib
python mnist_classifier.py
```

---

## 2. Backpropagation Algorithm

### Purpose
Implements a simple neural network trained using **Backpropagation** for logical problems like **XOR**, **AND**, and **OR**.

### Usage
```bash
pip install numpy
python backpropagation.py
```
